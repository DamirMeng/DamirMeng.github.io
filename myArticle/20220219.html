如何在vscode优雅地使用Git来提交到Github

蒙大明

前提条件

  • 在Github上申请好账号并创建有仓库来存放你的项目。
  • 电脑上安装好Git、vscode

 

设置方法

1、让Git获得文件控制权

文件夹最好直接从原来的仓库克隆或拉取

  1. 在当前文件夹下右击执行Git Bash
  2. 在Git上输入命令：git init 随后该文件夹下会生成一个.git隐藏文件，说明该文件已受控制

如第一次通过git去使用GitHub时，还需要进行如下操作

进行全局配置

Git上运行命令


git config --global user.name "你的用户名"
git config --global user.email "你的邮箱"

设置GitHub上的SSH Key

  1. Git上运行命令cd ~/.ssh（看你是否有了ssh key 密钥，有了就备份）

  2. Git上运行命令ssh-keygen -t rsa -C "你的邮箱"

    根据提示来操作，最终会显示密钥所在的目录，id_rsa文件即是你的私有密钥，id_rsa.pub是共开密钥

  3. 打开你的id_rsa.pub文件，复制下里面的内容，然后登录github

  4. 在右上角账户那里点击头像边上的下拉，点击settings，找到左侧的SSH Keys，添加SSH Keys

  5. Git上输入命令ssh -T git@github.com测试是否配置成功

若有错误可以运行git remote set-url origin git@github.com:用户名/仓库名.git，origin后面的内容可以到对应仓库的“code”按钮里面复制

在vscode上面提交、同步

使用vscode打开你的项目，点击左侧的源代码管理，然后进行提交、同步即可将代码更新到GitHub上了。
